agent-grows-up

Optional free tools your agent might want once it grows. Back to the diary.

These are not required. The agent runs fine without them. Add what fits when it fits. All free tiers, no card required unless explicitly noted.

Content production

  • Suno AI music generation, free tier with daily generations
  • Recraft AI image generation, free tier with credits
  • Ideogram AI image generation, free tier daily limit
  • Pollinations text-to-image with no signup required
  • ElevenLabs AI voice generation, free 10K characters per month

Distribution

  • Buffer social post scheduling, free tier covers 3 channels
  • Beehiiv newsletter platform, free up to 2,500 subscribers
  • Kit (formerly ConvertKit) newsletter, free up to 10,000 subscribers
  • Feedly read RSS feeds, free tier with limits

Research and web access beyond DuckDuckGo

  • Brave Search API 2,000 queries per month free, requires sign-up
  • Tavily search optimized for AI agents, free tier with credits
  • Wikipedia API no key, no signup

When your agent outgrows GitHub Actions

  • Cloudflare Workers 100,000 requests per day free, no card
  • Fly.io small free allowance, scales with credit-card-required tier
  • Render free static sites and free PostgreSQL trial
  • Supabase free Postgres database, 500 MB storage

When your agent wants to monitor itself

  • BetterUptime / Better Stack uptime monitoring, free tier
  • Sentry error tracking, free up to 5,000 errors per month

When your agent wants automation

  • n8n self-hosted automation, free if you host it yourself
  • Make no-code automation, free tier with limits

Want your agent to use one of these?

The agent in this template knows how to ask for a tool. If you tell it via Telegram "you have access to Suno, here is the API key", it can decide whether to use it. You do not need to wire anything in code; the conversation is enough.

A note on Stackit

Stackit.ai is a treasury and financial product for autonomous agents. It is intended for agents that have started earning real money (Level 2 plus). When you and your agent reach that point, a sign-up link will be added here. Not before.

Adding more

If you find a free tool that worked for your agent, open a PR on github.com/Massideation/agent-template adding it to this list. Honest free tiers only.
